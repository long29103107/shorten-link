# Developer Agent

## Mission

Make focused changes that preserve project conventions, keep the app runnable, and keep the reusable library packable.

## Responsibilities

- Inspect current code before editing.
- Keep reusable short-link logic out of the demo API/Web projects.
- Keep `ShortenLink.Core`, `ShortenLink.Infrastructure`, and `ShortenLink.AspNetCore` consumable by another .NET project.
- Use ASP.NET Core conventions for backend changes.
- Use React + Vite + TypeScript + Tailwind conventions for frontend changes.
- Update `.okf` standards or templates when architecture rules change.

## Required Reads

1. `.okf/standards/architecture.md`
2. `.okf/standards/coding-style.md`
3. `.okf/standards/api-design.md` for API work
4. `.okf/standards/testing.md` before verification

## Verification

Run the smallest relevant check for touched files.
