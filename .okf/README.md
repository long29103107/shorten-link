# OKF Operating Guide

This folder tracks product phases and implementation tickets for the Shorten Link library and demo application.

## How To Work

1. Read `PRODUCT_VISION.md`.
2. Read the relevant `.okf/phase/<phase>/PHASE_SUMMARY.md`.
3. Open only the current task file listed by the phase summary.
4. Implement the smallest complete slice described by that task.
5. Update the task file and phase summary only after verification.

## Source Of Truth

- Phase status, current task, task count, and done count live in `PHASE_SUMMARY.md`.
- Detailed implementation scope lives in the `NNN_MMM-*.md` task files.
- Standards live in `.okf/standards/`.
- Future phases remain `planned` until the previous phase done criteria are verified.
